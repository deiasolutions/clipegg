‚ú® Key Features

ü™∂ Ultra-Lightweight

Transmit only URIs and minimal metadata. Typical payload: 200-500 bytes vs 50KB-500MB for rich clipboard data.

üîó Reference-Based

Copy references to data, not the data itself. Perfect for large files, databases, and remote resources.

üéØ Context-Aware

Paste zones can filter by content type. Paste a product in a product list, a document in a file manager.

‚è±Ô∏è Time-Sensitive

Support for expiring links and temporary access tokens. Security built-in.

üîÑ Fallback Support

Graceful degradation to text/uri-list and text/html for non-ClipEgg applications.

üöÄ Framework-Agnostic

Pure JavaScript. Works with React, Vue, Angular, or vanilla JS. No dependencies.
